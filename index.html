Skwares

Your bookmarks, reloaded

This is an extension for the Chrome browser. If you have more than a few, handling your bookmarks with the built-in manager can be tedious. Who hasn't heaps of unsorted forgotten treasures down that hole ?

Skwares to the rescue !

And oh, it'd pronounced like "squares", of course, because your bookmarks are now shown as big square icons ;)

What to do ?

Move the skwares around : group them visually, organize them in a way that makes sense to you. Positions are remembered one session to another. Drop them on another tab to move them in the folders tree. Right-click to edit the tittle or remove the bookmark.

You like it ? Make it better !

  • Post issues, blames, praises, pieces of advices, wishes ...
  • Enroll as a contributor (see TODO.txt)

Install

Stable version in Google Chrome Webstore.

To run from this GitHub repo, you'll need to grab your own copy of jQuery, jQuery UI and jquery-context-menu and put all the things where they belong, see INSTALL.md
